# 📄 Automated Resume Parser

## Overview
This project is a Flask-based web application that automatically extracts candidate details from uploaded resumes.

## Features
- Upload Resume (.txt)
- Extract Name
- Extract Email
- Extract Phone Number
- Extract Skills
- Extract Education
- Resume Score
- SQLite Database
- Resume History
- Search Candidate
- Delete Candidate

## Technologies Used
- Python
- Flask
- HTML
- CSS
- Bootstrap
- SQLite
